# what it is:
hello and welcome to my first coding project: an interactive to-do list
of course, you can add and complete tasks, but after you complete them you also get points!
you can then exchange these points for rewards, that you can also personalize.
this will be my first entry to the Hackclub Sleepover event :D
parts of this code are closely inspired by "Brainrot-Academy"'s "PythonToDo" code on GitHub 

# how it works:
- enter "1" to add new tasks, you can view a list of them by entering "3"
- enter "2" to select the tasks you completed, after that you will receive a random number of points between 18 and 42 (averaging to 30)
- enter "4" to exchange the points for rewards (of which there are 3 levels, each costing 30/60/90 points; you can also personalize these tasks, for example the 30pt reward could be to grab a snack, while the 90pt reward to watch a movie)
- enter "5" to end the session

# how to run it:
- copy-paste the raw code into any terminal that supports Python, then run it
